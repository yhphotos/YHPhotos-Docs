# 欢迎使用 YHPhotos

欢迎来到 **YHPhotos** 文档站！这里是一个现代化、美观的文档展示平台，专门为展示您的 Markdown 文档而设计。

## 🎯 项目目标

YHPhotos 文档站旨在为用户提供一个：
- 📚 优雅的文档阅读体验
- 🎨 现代化的用户界面
- 🔧 易于使用和定制
- 📱 完美的移动端支持

## ✨ 特色功能

### 美观设计

- 支持明暗主题切换

### 用户体验

- 快捷键支持

## 💡 使用技巧

1. **搜索文档**: 使用顶部的搜索框快速找到需要的内容
2. **主题切换**: 点击右上角的主题按钮或在搜索框中按 `Ctrl/Cmd + D`
3. **移动端**: 在移动设备上，点击菜单按钮可以打开侧边栏
4. **快捷键**: 
   - `Ctrl/Cmd + K`: 聚焦搜索框
   - `Ctrl/Cmd + D`: 切换主题
   - `ESC`: 关闭侧边栏（移动端）

## 🔧 自定义说明

这个文档站可以轻松自定义：

- **样式**: 修改 `css/style.css` 文件
- **功能**: 编辑 `js/app.js` 文件
- **内容**: 在 `docs/` 目录下添加您的 Markdown 文件
