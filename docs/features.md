# 功能特性

YHPhotos 文档站提供了丰富的功能和特性，为用户提供优秀的文档阅读和管理体验。

## 🎨 界面设计

### 现代化视觉设计
- **Material Design 风格**: 采用谷歌的 Material Design 设计语言
- **优雅的排版**: 精心设计的字体搭配和行间距
- **精美图标**: 使用 Font Awesome 图标库提供丰富的图标支持
- **微交互**: 悬停效果、点击反馈等细致的交互设计

### 主题系统
- **双主题支持**: 明色主题和暗色主题自由切换
- **智能保存**: 主题选择自动保存到浏览器本地存储
- **一键切换**: 点击按钮或使用快捷键即可切换主题
- **系统适配**: 可根据系统设置自动选择主题

### 响应式布局
- **移动端优化**: 完美适配手机和平板设备
- **灵活网格**: 使用 CSS Grid 和 Flexbox 实现灵活布局
- **断点设计**: 精心设计的断点确保各种屏幕尺寸的最佳体验
- **触摸友好**: 针对触摸设备优化的交互区域

## 📚 文档功能

### Markdown 支持
- **完整语法**: 支持标准 Markdown 语法
- **扩展语法**: GitHub Flavored Markdown (GFM) 支持
- **代码高亮**: 多种编程语言语法高亮显示
- **表格支持**: 美观的表格样式和布局

### 内容展示
- **层次结构**: 清晰的标题层次和视觉引导
- **列表样式**: 美观的有序和无序列表
- **引用块**: 优雅的引用样式
- **链接处理**: 自动识别和优化链接显示

### 代码展示
- **语法高亮**: 使用 highlight.js 实现代码高亮
- **复制功能**: 一键复制代码块内容
- **行号显示**: 可选的行号显示
- **多种语言**: 支持 100+ 种编程语言

## 🔍 搜索与导航

### 智能搜索
- **实时搜索**: 输入即时显示搜索结果
- **模糊匹配**: 支持部分匹配和模糊搜索
- **搜索高亮**: 搜索结果中高亮匹配关键词
- **搜索历史**: 记录搜索历史（未来功能）

### 导航系统
- **侧边栏导航**: 清晰的文档结构导航
- **面包屑**: 显示当前位置和路径
- **快速跳转**: 快速跳转到任意章节
- **进度指示**: 显示阅读进度

### 快捷操作
- **键盘快捷键**: 丰富的快捷键支持
- **鼠标手势**: 支持鼠标滚轮和拖拽操作
- **触摸手势**: 移动端手势支持
- **无障碍**: 完整的键盘导航和屏幕阅读器支持

## 🔧 自定义与扩展

### 主题自定义
```css
:root {
    --primary-color: #6366f1;      /* 主色调 */
    --accent-color: #3b82f6;       /* 强调色 */
    --text-primary: #1f2937;       /* 主文本色 */
    --background-primary: #ffffff; /* 主背景色 */
}
```

### 布局配置
```javascript
const config = {
    sidebarWidth: 280,     // 侧边栏宽度
    theme: 'light',        // 默认主题
    autoGenerateNav: true, // 自动生成导航
    showLineNumbers: false // 显示行号
};
```
