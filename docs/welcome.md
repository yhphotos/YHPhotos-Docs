## 💡 一些小技巧

1. **找内容**: 顶部的搜索框是你的好帮手，输入关键词试试看
2. **切换主题**: 右上角有个小按钮可以切换深色/浅色模式，或者在搜索框按 `Ctrl/Cmd + D` 
3. **手机用户**: 点击左上角的菜单按钮可以打开导航栏
4. **键盘党福利**: 
   - `Ctrl/Cmd + K`: 快速定位到搜索框
   - `Ctrl/Cmd + D`: 一键切换深色/浅色模式

## 🔧 想要自定义？

当然可以！这个文档站就是为你准备的：

- **改样式**: 编辑 `css/style.css` 文件
- **加功能**: 修改 `js/app.js` 文件
- **写内容**: 在 `docs/` 目录下添加你的 Markdown 文件
